# teams-for-m1

Unofficial Microsoft Teams client for ARM Macs using [`Electron`](https://electronjs.org/).
It uses the Web App and wraps it as a standalone application using Electron.

Forked from IsmaelMartinez's work on ['teams-for-linux'](https://github.com/IsmaelMartinez/teams-for-linux).
